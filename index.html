Команды бота

Добавляет участника в таблицу.

Пример использования: /join IDN Имя Звание ДатаКМБ ДатаПовышения Отряд Специализации SteamID DiscordID Выговоры

Полный перезапуск бота.

Пример использования: /restarttab

Добавить выговор участнику.

Пример использования: /vygovor_plus Имя ПисьменныеУстные Причина

Убрать выговор у участника.

Пример использования: /vygovor_minus Имя ПисьменныеУстные Причина

Изменить звание участника.

Пример использования: /setrank Имя НовоеЗвание Причина

Подсчет баллов для проверяемого.

Пример использования: !под *состав* *повышения* *остальные_рапорты* *при нужде @пинг того кого проверяете*

Написать сообщение с упоминанием роли.

Пример использования: !написатьроль @роль текст

Написать сообщение без упоминания роли.

Пример использования: !написать текст
